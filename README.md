# Aron Amare Portfolio

## Overview

This is my personal portfolio website built using HTML, CSS, Bootstrap, and JavaScript. The website showcases my background, skills, projects, and professional journey as a Computer Science student and aspiring software developer.

## Features

* Responsive design using Bootstrap
* Dark mode toggle
* Typing animation effect
* Interactive profile image animation
* About Me page
* Contact page
* Skills and education sections
* Social media links

## Technologies Used

* HTML5
* CSS3
* Bootstrap 5
* JavaScript
* Git & GitHub
